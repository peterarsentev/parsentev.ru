Java EE. Разработка серверных приложений. Версия 2.0

Цели.

Я думаю, что большинство из вас уже искали подходящие вакансии для устройства на работу. Если вакансия написана продумана, то в требованиях в большинстве случаев будет указаны технологии из стека Java EE: Servlet, JSP, Spring, Hibernate, JMS, EJB, JNDI. Список этих аббревиатур можно продолжать до бесконечно и уровня хотелось работодателя.

В начале своей карьеры такие объявления меня тоже пугали. Я думал "как же надо много освоить", но с опытом, пришло понимание, как эти технологии построены и весь дискомфорт использования этих технологий пропал.

В этом курсе, я хочу научить вас, как правильно использовать данные технологии, рассказать о базовом принципе построения приложения на основе этих технологий. После курса, Вы сможете использовать любую технологии из стека JEE, даже если в курсе про нее не рассказано, зная базовый принцип построения, Вы с легкостью сможете овладеть этой технологией.

Все навыки полученные в курсы вы будите использовать в дальнейшем в своей работе, а так же при прохождении собеседований и выполнение тестовых заданий.

Формат и время проведения занятий.

В этом курсе я решил изменить способом и формат обучения. Пообщавшись с ученикам и прислушавшись к их советам. я решил попробовать новый формат. Теперь занятие будут в форме вебинаров. То есть в реальном времени, я буду рассказывать тему. показывать пример исполнения программы. Мне можно будет сразу задать вопросы и получить ответы. Ко всему курсу будет прилагаться методическое пособие с описанием решения домашних задачий. Эти материала остануться у вас на всегда и вы сможете обновить свои знания в любое время. Обучение будет в группах 5-6 человек. Занятия будет проходить в выходные дни по 3 часа с перерывом на 5-10 минут. В начале каждого занятие буду рассказываю про ошибки в исполнении домашних заданий. Длительность курса : 2 месяца.

Для прохождения курса слушатель должен владеть основами разработки на языке Java. После прохождения курса слушатель может претендовать на позицию middle java developer.

Программа курса.

  • Структура серверного приложения. Конфигурация Maven проекта под Web приложение
  • Протоколы передачи данных - HTTP, HTTPS. Взаимодействие клиент-сервер. Servlet, JSP.
  • Шаблон проектирования MVC на базе Servlet, JSP, JSTL.
  • Понятие CDI на базе фреймворка Spring (IoC, AOP)
  • Понятие CDI на базе технологии EJB (JNDI, J2EE Server, Session Bean, JMS, Message Driven Bean)
  • Интеграция с базой данных через JPA (Hibernate)
  • Интеграция с NoSQL базой данных MongoDB
  • Реализация MVC на базе Spring MVC
  • Реализация MVC на базе JSF 2.0
  • Интеграция с другими приложениями: RMI, JNDI, Web Service API, REST API, WebSocket, Email, JMS
  • Безопасность в серверных приложениях: Servlet Filter, Spring Security
  • Реализация SSO на базе интеграции с LDAP сервером
  • Клиентские языки: HTML, JS, CSS, Ajax(jquery)
  • Тестирование серверный приложений JMeter.
