Java от А до Я
=======================================
Обучающий курс
--------------

Автор
-----
Арсентьев П.С.

Цель
----
Цель курса - обучение слушателей теоретическим и практическим знаниям разработки на языке Java.
Курс начинается с основ синтаксиса языка и плавно переходит к темам серверной разработки.
Рассматриваются темы: Java Core, Maven, Git, JUnit, SQL, Servlet, JSP, Hibernate, Spring.
Каждое занятие курса содержит теоретический материал и практическое задание. После окончания курса слушатель сможет разрабатывать
сложные серверные приложения с использованиям популярных библиотек и фреймворков.

Данный курс является второй версией курса "Java. Путь от ученика до экспетра". На основании полученного опыта обучения, я дополнил материал,
а так же изменил некоторые темы. Уделил больше внимание тем проблемам с которыми сталкивались мои ученики.

Уровень слушателя
-----------------
Для прохождения курса слушатель должен иметь базовое представление о программировании в целом.
После прохождения курса слушатель может претендовать на позицию **junior java developer**.

План
----
Подготовительная часть. Инструменты разработки.
- Java
- Maven
- Tomcat
- PostgreSQL
- IDEA

Часть 1. Базовый синтаксис языка.
- Понятие JVM, Byte code.
- Структура класса. Класс. Поля. Методы.
- Типы данных. Примитивные и ссылочные типы.
- Условия
- Циклы
- Массивы
Часть 2. ООП
- Наследование.
- Инапсуляция.
- Полиморфизм.
- Абстрактные классы.
- Внутренние классы.
- Интерфейсы.

Часть 2.3. ООД.
- Принципы проектирования SOLID.
- Тестирование по методу черного яшика.

Часть 3. Коллекции
- Генерики
- Коллекция List
- Коллекции Set
- Коллекции Map
- Коллекции Tree
- Понятие интератор.

Часть 4. Память
- Понятие сборшика мусора.
- Типы сборщиков.
- Профилирование приложения.

Часть 5. Хранение данных
- Общие понятие про базы данных на основе PostgreSQL
- Базовый синтаксис SQL.
- JDBC
- Базовые понятие ORM. Hibernate.
- Конфигурирование Hibernate.
- Связь модельных представлений с объектами.
- Язык запросов HQL.
- Интеграционное тестирование.

Часть 6. Клиент-серверное приложение
- Общая схема клиент-серверного приложения.
- Протоколы HTTP, HTTPS.
- Servlet.
- JSP.
- Реализация MVC на Servlet, JSP, JSLT,
- Клиентские языки HTML, JS, CSS. Базовый синтаксис.
- Тестирование Mockito.

Часть 7. Модульное программирование на основе фреймворка Spring.
- Spring IoC.
- Spring HibernateTemplate.
- Spring MVC.
- Spring Security.
- Spring Test.
