Обучение и трудоустройство Java программистов от Junior до Senior.

Март 10, 2016 | Автор: Арсентьев Петр В данный момент я занимаюсь разработкой своего нового проекта – Сервис обучения и трудоустройства Java программистов от Junior до Senior. На первом этапе данный проект будет жить разрозненно в различных социальных сетях. Вторым этапом будет мигрированние в облако и доработка остальных сервисов. Про данный проект будет снято отдельное видео, в котором я подробно расскажу все возможности данного сервиса. Данная статья находится на редактировании и еще не окончена.

Реальный опыт в коммерческом проекте

Не секрет, что во всех вакансиях Java программистов требуется опыт работы от 1 года минимум. Для новичков данное требования является существенным барьером. Поэтому для вас я разработал уникальное предложение - участие в разработке проекта Job4j. Это даст вам возможность получить реальный опыт в коммерческом проекте. В проекте Job4j используеются передовые методики разработки программного обеспечения. Вы окунетесь во все этапы разработки, проектирования, тестирования, внедрения. Проект будет построен по архитектуре микросервисов. Вы познакомитесь с code review, continie intergration, pull requests, TDD, Agile. Стек тенхологий построен на базе JMV: Java, Scala.
Для участия в данном проекте нужно иметь минимальные знания про Serlvet, SQL, HTML, Ajax.

Как происходит процесс. Вам выдают задание. В описании к заданию будет информация, статьи и видео материал необходимый для выполнения задания. После выполнения задания происходит code review, проверяются ошибки в коде и тд. Более подробно про процесс будет рассказо в видео.

Сервис - обучение Java программированию

Проанализировав ситуацию в сфере обучения программированию, мы пришли к выводу, что основная проблема заключается в остуствии обратной связи от преподавателя об изученном материале. Сам ученик не в состоянии понять, правильно ли он выполнил задание или нет, так как одну и ту же задачу можно решить различными способами, и каждый из них будет иметь свои достоинства и недостатки. То есть данный процесс не может быть автоматизирован. В данном сервисе вы можете тренировать свои практические навыки программирования, получить оценку вашего кода от профессиональных программистов, улучшить свои знания в конкретных областях программирования.

Как проходит обучение в данном сервисе.
В системе существует три уровня сложности: Junior, Middle, Senior. Каждый уровень состоит из около 40 задач возрастающей сложности. Каждая задача посвящена конкретной теме из программирования. К задаче состалены список литераты и вопросы, необходимые для изучения и подготовки. В системе каждому ученику будет заводиться отдельная тема для каждой задачи и он будет назначаться отвественным. Когда ученик выполнил задание, он должен изменить отвественного на проверяющего и получить code review, ответ, замечения. После того как задание было одобрено, ученик получает следующее задание. Ученики в системе могут смотреть решение других учеников и давать комментарии к выполнению. Полезные комментарии будет оценнены баллами и в дальнейшем могут быть использованы для приобретения других услуг.
